# Muscle - 算法刷题助手

## 项目介绍

Muscle是一个专注于算法刷题的Web应用，旨在帮助用户系统性地学习和复习算法题目，提高刷题效率，为技术面试做好准备。

## 核心功能

- **计划模块**：日历视图管理学习计划，直观展示刷题进度和复习安排
- **练题模块**：提供清晰的题目展示和代码编辑环境，支持多种编程语言
- **复习系统**：基于艾宾浩斯记忆曲线的周期性复习机制，巩固学习成果
- **个人中心**：管理个人题解库和学习数据，追踪学习进度

## 技术栈

- 前端框架：Vue 3
- 状态管理：Pinia
- 路由管理：Vue Router
- UI组件库：Element Plus
- 图表可视化：ECharts
- 代码编辑器：CodeMirror
- 构建工具：Vite

## 开发指南


### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本
